AI Quiz Master
A modular Python application that leverages Generative AI to create interactive learning experiences.

🚀 Features

AI Question Generation: Dynamically creates MCQ quizzes using Gemini 2.0 Flash-Lite.

Secure User Auth: Custom authentication system using SQLite and SHA-256 password hashing.

Smart Theming: The UI automatically changes its primary color based on the quiz topic generated by the AI.

Resilient Architecture: Implements model fallback logic and rate-limit handling.

🛠️ Technical Stack

Frontend: Streamlit

AI Engine: Google GenAI SDK (Gemini API)

Database: SQLite3

Security: Hashlib (SHA-256)
